# RFC Index

This directory tracks design decisions for contract-facing FUSE changes.

## Lifecycle states

- `Proposed`
- `Accepted`
- `Rejected`
- `Implemented`
- `Superseded`

## Numbering

- Use zero-padded numeric prefixes: `NNNN-short-title.md`
- Example: `0001-module-import-aliasing.md`
- Keep numbers stable once merged

## Authoring

1. Copy `0000-template.md`
2. Fill all sections
3. Open PR with status `Proposed`
4. Update status as decision progresses

See `CONTRIBUTING.md` for criteria on when RFCs are required.
